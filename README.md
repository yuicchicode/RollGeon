# 🎲 RollGeon

**A tiny dungeon-flavored randomizer for indecisive adventurers.**

RollGeon is a playful wheel app made to help you randomly choose things when your party, your brain, or your snack instincts refuse to make a decision.

Need to decide what to eat? Roll it.  
Need to pick a movie? Roll it.  
Need to assign chores like a chaotic dungeon master? Absolutely roll it.

It is basically a magical tavern wheel with a medieval/D&D-inspired look — but without needing a wizard, a framework, or a bag of npm dependencies.

---

## 🏰 What is this?

RollGeon is just a simple, fun, browser-based roulette/wheel app.

You add options, press **SPIN**, and let fate choose for you.

That is it.  
No overcomplicated quest log.  
No cursed subscription scroll.  
No dragon guarding the settings menu.

Just a wheel, some options, and a little bit of chaotic destiny.

---

## ✨ Features

- 🎯 Add custom options to the wheel
- 🧙 Add emojis, colors, and weights
- ⚖️ Enable weighted rolls for uneven odds
- 🗡️ Elimination mode to remove winners after they are picked
- 🔊 Optional sound effects
- 🎉 Optional confetti celebration
- 📜 Recent roll history
- 💾 Local storage persistence
- 📤 Export your wheel
- 📥 Import a saved wheel
- 🎨 Randomize item colors
- 🏕️ Quick presets for food, movies, chores, and challenges
- 🏰 Pixel/medieval/D&D-inspired interface

## 🚀 How to run

Clone or download the project and open:

```bash
index.html
```

That is enough.

For the cleanest experience, you can also serve the folder locally:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

---

## 🎮 How to use

1. Add options to the wheel.
2. Customize emoji, color, and weight if you want.
3. Press **SPIN**.
4. Accept the ancient judgment of the wheel.
5. Blame fate, not yourself.

You can also press the **spacebar** to spin when you are not typing in a field.
